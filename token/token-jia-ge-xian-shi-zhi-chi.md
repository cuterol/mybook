# Token价格显示支持

### 背景 <a href="#bei-jing" id="bei-jing"></a>

显示Token价格是一项非常重要功能，DOMWALLET目前已支持每条链主流Token价格显示。同时DOMWALLET一直致力于和各个项目方合作提供更多Token价格可以被解析和显示。随着web3生态发展日新月异，不断有新的项目被创立，项目方Token资产未能及时被DOMWALLET支持价格。本文档将介绍不同类型项目方如何向TokenPocket提交价格支持申请。

### 已支持和未支持Token价格显示效果 <a href="#yi-zhi-chi-he-wei-zhi-chi-token-jia-ge-xian-shi-xiao-guo" id="yi-zhi-chi-he-wei-zhi-chi-token-jia-ge-xian-shi-xiao-guo"></a>

### TokenPocket Token价格数据来源 <a href="#tokenpocket-token-jia-ge-shu-ju-lai-yuan" id="tokenpocket-token-jia-ge-shu-ju-lai-yuan"></a>

TokenPocket价格解析有三大来源：

* 币市场：coinmarketcap、CoinGecko和Live Coin Watch等等
* 中心化交易所：coinbase、huobi和bian等等
* 去中心化交易：uniswap、pancakeswap和mdex等等

### 项目方申请TokenPocket价格显示支持 <a href="#xiang-mu-fang-shen-qing-tokenpocket-jia-ge-xian-shi-zhi-chi" id="xiang-mu-fang-shen-qing-tokenpocket-jia-ge-xian-shi-zhi-chi"></a>

对于未被TokenPocket支持其Token价格显示的项目方，如果希望TokenPocket支持其项目或者生态系Token价格显示，可以向TokenPocket提交申请，我们会在第一时间响应和支持。

根据Web3项目类型不同，需求也存在不同，我们大致将项目方需求分为三类。

#### **单Token项目方** <a href="#dan-token-xiang-mu-fang" id="dan-token-xiang-mu-fang"></a>

如果项目是一个专注于特定Web3方向，项目生态应该会发行一种或者多种Token。该类型项目方TokenPocket提交价格显示支持申请后，我们将在TokenPocket Token价格数据来源中寻找可用价格数据支持。

#### **DEX项目方** <a href="#dex-xiang-mu-fang" id="dex-xiang-mu-fang"></a>

如果项目一个去中心化交易所，那么该交易所存在很多交易对及其Token。该类型项目方为了提供更好用户体验，一般希望交易所下所有Token价格能够被钱包解析。可以向TokenPocket提交价格显示支持申请，通过合作支持项目方生态下Token价格。

#### **链项目方** <a href="#lian-xiang-mu-fang" id="lian-xiang-mu-fang"></a>

如果项目是一条链，该类型项目方为了推动链生态发展，一般希望原生币和链官方去中心化交易所Token被支持价格显示。可以向TokenPocket提交价格显示支持申请，深度合作，推动该链生态Token价格被TokenPocket支持。

#### **其他** <a href="#qi-ta" id="qi-ta"></a>

如果你项目不是以上三类，请联系我们，我们将不留余力去支持项目方Token价格显示。

#### **提交申请数据格式** <a href="#ti-jiao-shen-qing-shu-ju-ge-shi" id="ti-jiao-shen-qing-shu-ju-ge-shi"></a>

项目方需要提交官方网站地址、联系电话、邮箱地址等信息
